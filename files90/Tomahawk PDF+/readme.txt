Changes to Version 2.8.0
1. Added Korean language to user interface.

Changes to Version 2.7.7
1. Improved handling of foreign character sets in menu.
2. Changed the variable for printing the number of total pages. Before, if you specified for page numbers
   to start at 3 and you had 8 pages total, &p of &P would show the last page as 10 of 8. That has been
   corrected so that it will now display as 10 of 10 with 8 total pages and a starting page number of 3.
3. New Header/Footer variables, 'i' (current page) and 'I' (Total Pages): will display page numbers as Roman Numerals.
   ie: Page &i of &I would display 'Page VI of XII' if the current page is 6 and total pages is 12.
4. New 'Quick Format Painter' (Ctrl+Q) - works similar to the one in MS Word. See Example.tmd included with this download.

Changes to Version 2.7.6
1. Improved handling of foreign character sets.

Changes to Version 2.7.5
1. Improved look of menu - adapts to XP styles when present.
2. Fixed problem with images not loading correctly when opening html files
   that used '%20' in the path names.
3. Due to recent requests, TPdfPlus is completely portable now. 
   Does not store any settings to the registry.
   A. Spell check options are stored in file named 'Spell.cfg'
   B. Language Setting is stored in file named 'TPdfPlus.lng'
   C. Most recently used files are stored in file named 'TPdfPlus.ini'

Changes to Version 2.7
1. Simplified 'File Open' operation. Added more text based file types that will
   open in the editor.
2. Import tables directly into Tomahawk documents from delimited files (Comma, 
   Tab, etc.)
3. View, edit, format and print multi-sheet Excel spreadsheets. Save and "New 
   Spreadsheet" in Corporate version only.
   Full support for most common formulas and sorting in spreadsheet module
4. Convert Excel spreadsheets to PDF natively. Note - supports spreadsheets for
   Excel 97 or greater. No support for embedded objects or macros. Excel does not
   have to be installed.
5. Minor improvements.


Changes to Version 2.6
1. Tomahawk PDF+ no longer creates a temporary file before converting to PDF.
2. Page header and footer info are now used when converting to PDF.
3. PDF conversion screen now reflects the margins used in the Page Setup screen.
4. Turned margin clipping off in Printing unit. This allows Header and Footer info to 
   export correctly when printing to Virtual Print Drivers such as Adobe Distiller
   and FinePrint. Corporate version allows this to be toggled on/off.
5. Improved Image Manager.
